### 1. 가상환경
#
여러가지 프로젝트들이 다른 버전의 모듈을 사용할때   
한개의 환경위에서 충돌로 인해 에러가 발생할 수 있다.  
그래서 가상환경을 만듦으로써 다른 버전의 모듈들을 적용시킬수 있도록 한다.

<br/>

### 2. 가상환경 생성하기
#
```
python3 -m venv python_basic
```

가상환경을 만들면 새로운 파일들이 생긴다.
- bin
- include
- lib

<br/>

### 3. 가상환경 실행하기
#
```
source bin/activate
```
